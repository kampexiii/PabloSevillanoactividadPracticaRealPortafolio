# Portafolio personal - Actividad práctica real

Este repositorio es para la práctica de clase de 2º DAW.

---

## Pasos que he seguido

1. **Crear repositorio**  
   Hice un repo en GitHub con el nombre `PabloSevillanoactividadPracticaRealPortafolio`.

2. **Clonar en local**  
   Lo cloné en mi ordenador con `git clone`.

3. **Archivos iniciales**  
   Creé tres archivos básicos:

   - `index.html` con un título y un párrafo.
   - `style.css` con estilos muy simples (fuente y color del título).
   - `script.js` que muestra un mensaje en la consola.

4. **Primer commit**  
   Subí estos archivos al repositorio con el primer commit.

5. **Crear rama `contacto-form`**  
   Hice una nueva rama para añadir un formulario de contacto.

6. **Añadir formulario**  
   En el `index.html` puse un formulario muy básico con nombre, email y mensaje.  
   Hice dos commits en esa rama (uno para el formulario y otro para cambiar el texto del botón).

7. **Pull Request y merge**  
   Subí la rama a GitHub, abrí un Pull Request y lo mezclé con `main`.

8. **Actualizar local**  
   Volví a la rama `main` en mi ordenador y actualicé con `git pull`.

9. **Crear Issue**  
   Abrí un issue en GitHub llamado “Añadir validación de formulario” para dejar pendiente la validación de los campos.
